La Historia de la Web

Un invento que revoluciono nuestras vidas

Que es la Web?

La World Wide Web es un sistema global diseñado para gestionar y compartir información a través de Internet.

Aunque solemos usar ambos términos como sinónimos, es importante diferenciarlos: Internet es la red física de cables y conexiones, mientras que la Web es el servicio que nos permite visualizar el contenido que viaja por esa red.

Su funcionamiento se basa en lo que llamamos hipertexto, que no es más que la capacidad de conectar documentos entre sí mediante enlaces o links. Esto permite que la navegación sea libre y no lineal; el usuario decide qué camino seguir al saltar de una página a otra.

Hoy en día, la Web es el medio principal por el cual accedemos a prácticamente todo el conocimiento humano, servicios y entretenimiento, permitiendo que archivos de texto, imágenes y videos estén disponibles de manera inmediata en cualquier parte del mundo.

Como se origino la Web?

La Web fue concebida en 1989 dentro del CERN, el laboratorio de física de partículas más importante del mundo, ubicado en Suiza.

Su creador fue el ingeniero británico Tim Berners-Lee, quien observó que los científicos tenían serias dificultades para intercambiar información porque sus datos estaban atrapados en computadoras aisladas con sistemas diferentes. Para solucionar este caos, Berners-Lee desarrolló tres pilares técnicos: un lenguaje para escribir las páginas (HTML), un sistema para darles una dirección única (URL) y un método para que las computadoras se comunicaran entre sí (HTTP).

Sin embargo, el factor que realmente impulsó su éxito ocurrió en 1993, cuando el CERN tomó la decisión de liberar el código de la Web para que fuera gratuito y de dominio público.
