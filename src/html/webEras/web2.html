Que es la Web 2.0

La Web 2.0 (o Web Social) surgió alrededor de 2004 y cambió las reglas del juego. Pasamos de ser simples lectores a ser creadores.

En esta etapa, el internet se volvió dinámico, colaborativo y centrado en el usuario. Si tienes una cuenta de Instagram, TikTok o Wikipedia, estás viviendo en la Web 2.0.
